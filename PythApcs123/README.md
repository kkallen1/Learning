## ch1

- 多行指令可以用 `;` 分隔，來放在同一行
  ```py
  a = 5; b = 10
  ```

- 將變數的值互換
  - 互換的寫法:
    ```py
    a, b = b, a
    ```
  - 沒辦法將 a 跟 b 的值交換的寫法:
    ```py
    a = b
    b = a
    ```

- `print()` 預設是以空白分隔的，可以用 `sep` 參數指定其他分隔符號

## ch2 & ch3

- 算式不要寫得太複雜，避免可讀性變差

- 小數點的計算要特別小心
  - e.g. `0.1 + 0.2` 可能會得到 `0.30000000000000004`
  - 可以使用 `round()` 函數來四捨五入
